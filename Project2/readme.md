Titanic Survival Prediction and Exploratory Data Analysis

This project performs an in-depth exploratory data analysis and builds a machine learning model to predict passenger survival on the Titanic. 
The dataset contains passenger information such as age, gender, ticket class, fare, and family relations, which are used to analyze patterns related to survival.
The project begins with data cleaning and preprocessing, including handling missing values, removing redundant columns, and converting categorical variables into 
numerical formats suitable for modeling. Exploratory Data Analysis (EDA) is then conducted to understand the relationships between different features and the survival
outcome through statistical summaries and visualizations.
Key insights were obtained by analyzing how factors such as passenger class, gender, age, and fare influenced survival probability. Visualization techniques were 
used to explore distributions, correlations, and trends within the dataset.
After the analysis phase, a machine learning model was developed to predict whether a passenger survived or not. The dataset was split into training and testing sets,
and classification algorithms were applied to train the model and evaluate its performance.
This project demonstrates the complete data science workflow, including data exploration, feature selection, preprocessing, model training, and evaluation using Python.

Technologies Used

Python
pandas for data manipulation
NumPy for numerical operations
Matplotlib and Seaborn for data visualization
scikit-learn for machine learning model development
